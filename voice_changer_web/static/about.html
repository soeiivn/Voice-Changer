About VoiceChanger Pro

一个完全基于 DSP 的实时语音变声器，支持风格互斥 + 空间音效叠加，纯浏览器运行。

核心技术

  • 音调音色转换
    PSOLA + Formant Envelope 实现实时 pitch / formant 调整
  • 空间音效（Schroeder Reverb）
    输入 → 4× Comb Filters（并联） → 2× Allpass Filters（串联）
    梳状滤波器形成混响尾音，扩散滤波器增加反射密度
  • 特殊音效
    • 电话：带通滤波 + 轻微失真
    • 机器人：环形调制 + 低通滤波
    • 卡通：Pitch Shift + Formant + 非线性增强
  • 实时架构
    FastAPI + WebSocket + AudioWorklet（超低延迟）

核心特性

  • • 7种预设风格 + Custom 自定义模式
  • • 风格与特殊音效互斥，空间音效可自由叠加
  • • Echo Strength 实时强度调节
  • • Pitch / Formant 实时滑块控制
  • • 锁控 UI（与 PC 端一致）
  • • 电脑浏览器即用或安装 PC 端 APP
